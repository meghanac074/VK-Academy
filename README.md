# VK Academy – Course Website

This is a responsive educational website developed for **VK Academy**, an academic coaching platform. It showcases the academy's courses and pricing, and provides a user-friendly interface for prospective students.

## 📁 Project Structure

```
courseproject/
├── index.php               # Home page
├── price.php               # Course and pricing information
├── photo.php               # Gallery or photo section
├── thankyou.php            # Confirmation or thank you page
├── p1.jpg, p2.jpg, p4.avif # Media assets used in the site
```

## 🛠️ Technologies Used

- **HTML5 / PHP** – For dynamic content rendering.
- **CSS3** – For styling and responsiveness.
- **Bootstrap (if included)** – For mobile-first responsive design (verify in code).
- **Images** – Used to improve visual engagement.

## 💡 Features

- ✅ **Responsive Design** – Works on desktop, tablet, and mobile devices.
- ✅ **Clean & Readable Code** – Easy to maintain and extend.
- ✅ **Course & Pricing Information** – Accurate as per the PDF provided by VK Academy.
- ✅ **Image Gallery** – Showcases the academy’s environment or events.
- ✅ **Thank You Page** – A confirmation page after form submission (if applicable).

## 🚀 How to Run

To run this project locally:

1. Make sure you have [XAMPP](https://www.apachefriends.org/) or any PHP server installed.
2. Move the `courseproject` folder into your web root:
   - For XAMPP: `C:\xampp\htdocs\courseproject`
3. Start Apache from the XAMPP control panel.
4. Open your browser and go to:
   ```
   http://localhost/courseproject/index.php
   ```
